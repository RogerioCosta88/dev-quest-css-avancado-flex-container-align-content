Align-content

O align-content alinha os elementos no eixo-transversal quando tem espaço no container. ATENÇÃO: se houver poucos elementos no container, devemos estilizar o container com flex-wrap: wrap, ficará assim:
.div-do-container {
        flex-wrap: wrap;
}
Para que visualmente fique mais claro.
